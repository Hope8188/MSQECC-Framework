# MSQECC Engineering Superpowers (Frontend Architecture Rules)

## 1. The "Taste-Skill" Foundation
- **No Vercel-blandness.** Avoid the default startup look.
- **Quantum Brutalism.** Emphasize raw, exposed data, raw physics math, large stark typography, monochromatic themes with tactical, high-contrast accents (e.g., #00FF41, #FF0055).
- **Pixel Perfection.** Every margin, padding, and border radius must have a specific, justifiable rhythm (e.g., strict 4px/8px/16px/32px scale).

## 2. Gemini Superpowers for UI Engineering
*Activating Supreme Agentic Traits:*
- **Micro-Interactions are Mandatory:** A button is not just a button. It has a hover state, active state, focus ring, and transition timing (`ease-out`, 200ms).
- **Custom Cursor Dynamics:** Default OS cursors pull users out of the immersive experience. We must implement a custom pointer (e.g., a glowing dot with an echoing magnetic ring) to emulate observing quantum states.
- **Scroll Hijacking (Lightly) & Parallax:** Content shouldn't just scroll; it should reveal itself dynamically via `framer-motion` (viewport triggers, staggered delays).
- **Mathematical Layouts:** Use CSS Grid extensively to map out UI elements like a tensor network. Borders should be crisp (1px solid rgba(255,255,255,0.1)).

## 3. Cursor Rules & Code Standards
Whenever modifying the React/Vite stack:
1.  **TypeScript Strictness:** Never use `any`. Always type props clearly.
2.  **Tailwind Arbitrary Values:** Exploit Tailwind's arbitrary values for precise physical measurements `w-[1px]` over `w-px` when specific aesthetic control is needed.
3.  **Framer Motion Mastery:** Use `AnimatePresence` for unmounting. Modulate `ease: [0.16, 1, 0.3, 1]` for luxurious hardware-accelerated cubic-beziers. 
4.  **Lucide React Icons:** Use strictly 1px or 1.5px stroke geometries to mimic architectural blueprints.

## 4. Advanced Animation & 3D Topologies (GSAP + Three.js)
1. **Scroll Linearization (Lenis):** The DOM must not stutter. Wrap the root app in `<ReactLenis root>` to ensure layout ticks are unified perfectly with the monitor refresh rate for GSAP plugins.
2. **GSAP Orchestration (`@gsap/react`):** Only use `useGSAP()` for React component isolation. Use ScrollTrigger for elements that require scrubbed (timeline-attached) reveals or complex multi-stage staggering that Framer struggles with.
3. **React Three Fiber (R3F):**
    - The `1/D^2` noise matrix must be represented as a literal 3D point cloud or interconnected node graph. 
    - Use `useFrame` strictly for non-reactive topological mutations (rotating the galaxy model, pulsing entanglement connections).
    - Canvas must have `gl={{ antialias: true, alpha: true }}` to composite seamlessly over the HTML tensor dot-grid.
