# CSR Verification OnePager

## What CSR is
CSR is a research program attempting to derive galaxy-scale gravitational behavior from a deeper operator-and-measure structure instead of fitting galaxy curves directly.

## What is proved
- an internal bridge result exists
- an internal `T^3` lattice renormalization result exists
- a scale-invariant template can reproduce the right low-acceleration scaling pattern structurally
- the deep-IR theorem target is explicit: any real derivation must land on the equivalent of `q = 3`

## What is not proved
- no derived CSR galaxy law exists
- no branch is scientifically live under the strict suite
- no hardware result on this machine proves CSR physics

## Best current branch
The best-performing branch is still the scale-invariant template.

Current result on the direct observed main gate:
- aggregate `R^2 = 0.8776`
- median `R^2 = 0.6709`
- beats baryons on `2/3` galaxies
- mean residual lag-1 autocorrelation `0.8273`

Why it is still not accepted:
- it is a template, not a derivation
- the ablated version is effectively unchanged
- it does not clear the current evidence-over-baryons threshold
- its residuals are still highly structured rather than close to white noise

## New derivation-first branch
A new constraint-derived root law was added to test whether the square-root bend can be obtained analytically without a hand-wired MOND interpolation function.

Result:
- direct observed aggregate `R^2 = 0.8209`
- direct observed median `R^2 = -6.2341`
- beats baryons on `3/3` galaxies
- mean residual lag-1 autocorrelation `0.8891`

What this means:
- deriving only the closure is not enough
- the quotient normalization itself still has to be derived

## What the new theorem audit says
The current `T^3` backbone does not yet force `q = 3`.

It supports:
- bridge structure
- lattice renormalization geometry

It does not yet support:
- a derived deep-IR exponent
- a derived transition scale

So `q = 3` is still the correct target, not yet the achieved result.

## Main gate
The main adversarial external gate is the direct observed 3-galaxy set:
- `m33`
- `n1560`
- `n5533`

These are local machine-readable observed rotation curves with baryonic decomposition.
This is enough to falsify weak branches, but not enough to validate a theory beyond reasonable doubt.
