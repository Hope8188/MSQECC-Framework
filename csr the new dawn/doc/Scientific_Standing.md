# Scientific Standing

## Direct Verdict

MSQECC is not a theory of everything.

It is currently:

- a nontrivial galaxy phenomenology programme,
- a real `T^3` toric-code backbone,
- and a narrowed matter-coupling problem with one viable noncommuting syndrome-channel route.

## Strongest New Result

The cleanest new theoretical result is the syndrome-channel split:

- commuting `Z_edge tensor sigma_z` coupling fails,
- noncommuting `X_edge tensor sigma_z` coupling succeeds,
- exact coherence-decay fits match the predicted `2 g^2 / kappa` law.

That is the first genuinely useful code-to-matter mechanism result in the programme.

## Strongest New Constraint

The strongest new empirical-theory constraint is the positivity audit:

- the fitted galaxy ansatz often implies negative effective density,
- and no single global positive-density `k` exists for the full SPARC sample.

So the current law is not physically admissible as a literal positive dark-density model.

## External Status

The external picture is mixed.

- the frozen SPARC model is positive in the aggregate on the new non-SPARC sample,
- but median per-galaxy performance remains negative,
- and MOND remains stronger.

So the empirical core survives, but external validation is incomplete.

## Bottom Line

The programme becomes serious only if the noncommuting syndrome channel can be justified from the full code and the galaxy law can be reformulated to survive positivity and external testing.

That is the real frontier now.
