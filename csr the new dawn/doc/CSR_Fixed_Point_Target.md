# CSR Fixed-Point Target

## Purpose
This note replaces vague "derive MOND somehow" language with a typed theorem target.

The question is not whether a template can imitate the low-acceleration bend.
The question is whether the internal `T^3` renormalization and bridge program can force the deep-IR scaling needed for that bend.

## Minimal target
Write the deep-IR action density as:

`L_IR ~ |grad(phi)|^q`

Then the equivalent deep-IR observables are:

- `mu(x) ~ x^(q-2)`
- `g ~ g_N^(1/(q-1))`

where `x = |grad(phi)| / a_*`.

## Why `q = 3` is the target
If the galaxy law is MOND-like in the deep regime, then:

- low-acceleration RAR slope must be `0.5`
- low-acceleration MDAR slope must be `-0.5`
- asymptotically flat rotation curves require zero residual radial exponent

Those three constraints are simultaneously satisfied only when:

- `q = 3`
- `mu(x) ~ x`
- AQUAL free-energy density scales as `y^(3/2)` with `y = |grad(phi)|^2 / a_*^2`

This is not yet a derivation from CSR.
It is the correct target any derivation must hit.

## What this means operationally
The next theorem-level task is:

1. derive the deep-IR operator scaling from the actual `T^3` program
2. show that it forces `q = 3`, or an exactly equivalent low-acceleration law
3. derive the transition scale `a_*`

If the fixed-point program does not force `q = 3` or an equivalent target, then the current CSR galaxy-law route is not a derivation.

## What this does not mean
It does not mean:

- `q = 3` has been derived already
- the current template is validated
- the bridge is already connected to galaxy dynamics

It means only this:

the current route needs a fixed-point theorem, not another fit family.

## Bottom line
The next legitimate CSR milestone is not better interpolation.
It is a fixed-point derivation of the deep-IR target.
